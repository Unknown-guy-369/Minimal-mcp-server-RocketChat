## 🛠️ Getting Started

### 1. Install Dependencies

```bash
cd 
npm install
```

### 2. Configure Environment

Rename `.env.example` to `.env` and set your Rocket.Chat credentials:

```env
ROCKETCHAT_URL=http://localhost:3000
ROCKETCHAT_AUTH_TOKEN=your_personal_access_token
ROCKETCHAT_USER_ID=your_user_id
```

### 3. Build & Run

```bash
npm run build
npm start
```

### 4. Inspect with MCP Inspector

```bash
npx @modelcontextprotocol/inspector node --env-file=.env dist/server.js
```
